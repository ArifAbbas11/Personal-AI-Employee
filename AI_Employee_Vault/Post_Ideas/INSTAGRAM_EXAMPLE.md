## How to Use This Template

1. **Add Your Image:**
   - Place your image file in this same folder (Post_Ideas/)
   - Supported formats: .jpg, .jpeg, .png
   - Example: `demo_screenshot.jpg`

2. **Update the Image Field:**
   - Change `Image: demo_screenshot.jpg` to your actual filename
   - The system will automatically find it in this folder

3. **Write Your Caption:**
   - Keep it under 2,200 characters
   - Use emojis to make it engaging
   - Add relevant hashtags (up to 30)

4. **Move to Approved:**
   - When ready, move this file to `Approved/social_media/`
   - The system will post within 60 seconds

## Tips for Instagram Posts

- **Image Quality:** Use high-resolution images (1080x1080 or 1080x1350)
- **Caption Length:** First 125 characters are most important
- **Hashtags:** Mix popular and niche hashtags
- **Emojis:** Use 3-5 emojis for engagement
- **Call to Action:** Ask questions or encourage comments

## Workflow

1. Create your post file here: `Post_Ideas/MY_POST.md`
2. Add your image to: `Post_Ideas/my_image.jpg`
3. Reference it: `Image: my_image.jpg`
4. Move to: `Approved/social_media/MY_POST.md`
5. System posts automatically!
